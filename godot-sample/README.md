# Godot 4.6 Sample Project

A fully working Godot 4.6 demo showcasing all assets generated by pixel-asset-gen.

## Setup

1. Generate the assets from the parent directory:

```bash
cd ..
python generate_assets.py --clean --gif --gif-scale 8
```

2. Copy generated assets into this project:

```bash
# From the repo root
cp -r assets godot-sample/assets
```

3. Open in Godot 4.6:
   - Open Godot Engine 4.6
   - Import this folder as a project
   - Run the main scene (`scenes/world.tscn`)

## Controls

| Key | Action |
|-----|--------|
| Arrow Keys | Move |
| Shift | Run |
| Space | Attack (plays weapon animation) |
| Q | Previous weapon |
| E | Next weapon |

## What's Included

### World (50x40 tiles)
- Village with buildings at proper scale
- Castle courtyard with stone floor
- Forest with trees, rocks, grass
- Water pond with animated waves
- Lava pool near dungeon entrance
- Dirt paths connecting buildings
- Sand beach near water

### Player
- 7 animation states (idle, walk, run, attack, jump, hit, death)
- 4 directions each
- Visible held weapon sprite
- Weapon switching (Q/E) with 7 weapons
- Weapon attack slash animations

### Enemies (10 instances)
- Slime, skeleton, bat, ghost, goblin
- Wandering AI with random movement
- Idle, hit, and death animations
- Scattered across adventure zones

### Buildings (21 structures)
- Village row: houses, inn, blacksmith, church, barn
- Castle: main castle, watchtowers, armory, barracks
- Shops: market stalls, merchant tent
- Adventure: dungeon entrance, lighthouse, ruins, crystal cave
- Animated: chimney smoke, forge fire, windmill spin, fountain

### Weapons
- Weapon rack display in castle courtyard
- Shield display
- Rarity weapons (legendary, epic)
- Enchanted weapons scattered in world (fire, ice, lightning, poison)

### Effects
- Campfire with smoke
- Magic sparkle, heal aura
- Portal, lightning strike
- Energy orb, shield bash, frost burst

### NPCs
- Villagers wandering in village
- Merchant near market stall

## Viewport

- 1280x720 HD landscape
- 1.5x camera zoom
- Nearest-neighbor texture filtering (pixel-perfect)
